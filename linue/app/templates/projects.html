Lists, Tuples, Dictionaries

tupleExample = ('Carrot', 'Broccoli', 'Potatoes')
dictExample = {'1':'A', '2':'B', '3':'C'}

print(listExample)
print(tupleExample)
print(dictExample)

print(listExample[0])
print(tupleExample[1])
print(dictExample['3'])

Conditional, Comparison, and Logical Operators

Conditional operators: if, elif, else
Comparison operators: ==, !=, >, >=, <=
Logical operators: and, or, not

age = 13

if (age < 1):
    print('You are not old enough to learn programming')
else:
    print('You are old enough to learn programming!')

if ((age != 0) and (age <= 18)):
    print('Learn programming while you are still young!')
elif ((age == 19) or (age >= 20) and not (age > 99)):
    print('You are never to old to start programming!')
else:
    print('Your age seems to be out of range')

Try changing the age variable and see what you can get!
